The Piano
=========

This example shows a piano on screen that players can
play with their keyboard. It uses pre-made sound files
for every piano key.

Extended by: Cameron Petras & Matthew Petras

Instructions: Press Run and use top two rows of keys/mouse to activate piano.

Features: Visualizer now with completely random color and vertical placement. Ovals fade away over time.
